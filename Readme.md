# Django ChatGPT API Docker

## Overview
This project is a Dockerized version of a Django-based API service that interacts with the OpenAI ChatGPT model to provide responses to user messages.

## Build and Run
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/danou294/OPENAI-API.git
   ```
2. **Build the Docker Image:**
   ```bash
   docker build -t my_django_app .
   ```
   
3. **Run the Docker Container:**
   ```bash
   docker run -p 8000:8000 my_django_app
   ```
   
4. **Make Requests:**

Send POST requests to http://127.0.0.1:8000/chat/ with JSON payload:

   ```bash
    {
      "message": "Your message here"
    }
   ```
That's it! Your Django ChatGPT API service is now running inside a Docker container. You can send messages to it and receive responses generated by the ChatGPT model.
